# 野崎タイマー

タイマーを表示する拡張です。
母港アクセス時に残り時間ボイス通知を行い、15分以上経過で自動リセットします。

---

## 仕様

- 表示: 別窓（拡張の小型ウィンドウ）に常駐表示
  - リセットボタン
  - 残りタイマー（分:秒）
  - ボイス ON/OFF
- タイマー:
  - リセットボタン = 開始/リセット
  - 一度起動するとカウントは常に進む
  - 母港通信時に15分経過している場合にタイマー自動リセット
  - 編成画面の操作は検知しないので野崎を手動編成した場合はタイマーも手動リセットが必要
- 音声:
  - 母港アクセス時に判定
    - 15分経過: 「リセット」ボイス + タイマー0に戻す
    - 未経過: 「残り◯分」ボイス
  - ボイス ON/OFF で切替

---

## インストール

### ダウンロード

1. このページ右上の **緑の「Code」ボタン** をクリック  
2. **「Download ZIP」** を選択  
3. ダウンロードしたZIPを解凍  

### Chromeへの読み込み

1. Chromeで `chrome://extensions/` を開く  
2. 右上の「デベロッパー モード」をオン  
3. インストール
  - 「パッケージ化されていない拡張機能を読み込む」  > 解凍したフォルダを選択 
  - またはフォルダをドラッグ
4. 拡張を再読み込み後、別窓が自動表示されます（閉じた場合は拡張アイコンで再表示）

---

## 使い方

- 初回: 野崎を旗艦 or 母港リセット時にタイミングを合わせて「リセット」で開始
- 野崎を編成した場合は都度「リセット」でタイマー開始/リセット
- 「ボイス ON/OFF」で音声通知の切替
- 閉じた場合は拡張アイコンで再表示
